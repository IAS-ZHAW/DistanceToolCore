1. smallset.isis.dat
Set of 2197 records with 135 key values each. Eac key value is an integer number

Format:
record_id number_of_non_zero_descriptors descriptor_id1 descriptor_value1 descriptor_id2 descriptor_value2 ...

all not explicitly mentioned descriptors have the value 0

5. smallset.activity
Gives the information about the biolical targets and their classification for each record in smallset
Each record is associated with one or mor biological targets, which is classified in hierarchical target classification system
Format:
record_id\ttarget1:[target_class1:[target_superclass1]...][\ttarget2:[target_class2:[target_superclass2]...]]...
Example:
---------------------------------------------------------------------------------------------
140614	neprilysin: metalloendopeptidase: peptidase: hydrolase: enzyme:		peptidyl-dipeptidase a: peptidyl-dipeptidase: peptidase: hydrolase: enzyme: 
---------------------------------------------------------------------------------------------
